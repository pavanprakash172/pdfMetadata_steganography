# pdfMetadata_steganography
This project explores how hidden messages can be embedded inside PDF metadata using Python. Instead of altering the visible content of a document, the secret information is stored within metadata fields such as Title or Author.


## Overview
PDF files contain more than just visible text and images — they also store background information known as metadata. This project demonstrates how those metadata fields can be used to hide and retrieve secret messages without changing the actual content of the document.
The goal of this project is to understand the fundamentals of steganography through a practical demonstration of data hiding techniques using Python.


## Features
- Embed secret messages in PDF metadata
- Extract hidden messages from PDF files
- Works with existing PDF documents


## Technology Stack
- Python 3.x
- PyPDF2
- argparse
- Jupyter Notebook


## Installation
Clone the repository:
```bash
git clone https://github.com/pavanprakash172/pdfMetadata_steganography.git
cd pdfMetadata_steganography
```

Install dependency:
```bash
pip install PyPDF2
```


## Usage
1. Open Jupyter Notebook   
2. Open `PDFmetadata_steganography.ipynb`  
3. Run the cell to:
   - Embed a secret message
   - Extract the hidden message
